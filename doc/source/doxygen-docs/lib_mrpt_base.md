smaller libraries for the sake of modularity of easy of code reusability:

 - \ref mrpt_config_grp
 - \ref mrpt_expr_grp
 - \ref mrpt_db_grp
 - \ref mrpt_img_grp
 - \ref mrpt_io_grp
 - \ref mrpt_math_grp
 - \ref mrpt_random_grp
 - \ref mrpt_rtti_grp
 - \ref mrpt_serialization_grp
 - \ref mrpt_system_grp
 - \ref mrpt_typemeta_grp
