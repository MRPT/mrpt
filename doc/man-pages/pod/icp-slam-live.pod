=head1 NAME

icp-slam-live - ICP-based SLAM using scan matching with a LIDAR in real-time

=head1 SYNOPSIS

icp-slam-live I<config_file.ini>

=head1 DESCRIPTION

B<icp-slam-live> is a command-line application which connects to any 
MRPT-supported LIDAR and creates a map using simple scan matching.
