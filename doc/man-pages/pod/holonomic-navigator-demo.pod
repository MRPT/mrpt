=head1 NAME

holonomic-navigator-demo - GUI application for robot reactive navigation

=head1 SYNOPSIS

holonomic-navigator-demo

=head1 DESCRIPTION

A GUI program to demonstrate, test and experiment the usage of the mrpt-nav
library regarding the navigation of holonomic robots, that is, 
those which can move freely in any direction. These methods do not account
for the shape of the robot, i.e. it's a point or a circle.
