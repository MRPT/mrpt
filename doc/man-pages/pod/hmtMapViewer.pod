=head1 NAME

hmtMapViewer - Robotic Hybrid Metric Topological (HMT) map viewer

=head1 SYNOPSIS

hmtMapViewer

=head1 DESCRIPTION

B<hmtMapViewer> is a GUI application that allows loading and visualizing
complex HMT maps for mobile robotics applications.
