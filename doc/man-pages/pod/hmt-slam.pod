=head1 NAME

hmt-slam - Robotic Hybrid Metric Topological (HMT) mapping CLI

=head1 SYNOPSIS

hmt-slam

=head1 DESCRIPTION

B<hmt-slam> is a CLI application which allows building complex HMT maps.
