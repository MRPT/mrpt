=head1 NAME

2d-slam-demo - A didactical GUI application to learn how 2D SLAM works

=head1 SYNOPSIS

2d-slam-demo 

=head1 DESCRIPTION

B<2d-slam-demo> is a GUI application which allows the user to run a Extended
Kalman Filter (EKF) implementation of range-bearing 2D SLAM in step-by-step or 
continuous mode. Graphs are shown for the observations, data association, etc...
