=head1 NAME

pf-localization - MonteCarlo mobile robot localization

=head1 SYNOPSIS

pf-localization I<config_file.ini>

=head1 DESCRIPTION

B<pf-localization> is a command-line application which processes a rawlog and 
tracks a mobile robot pose from the sequence of odometry and observations. It can also 
perform global localization.
