=head1 NAME

SceneViewer3D - 3D robotic scenes viewer

=head1 SYNOPSIS

SceneViewer3D [I<file.3Dscene>]

=head1 DESCRIPTION

A viewer of 3D scenes generated by many other MRPT applications, in the 3Dscene format: a 
(possibly gz-compressed) file where a mrpt::opengl::COpenGLScene has been dumped to.
