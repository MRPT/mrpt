=head1 NAME

simul-landmarks - Robot dataset simulator

=head1 SYNOPSIS

simul-landmarks I<config_file.ini>

=head1 DESCRIPTION

B<simul-landmarks> is a command-line application which generates a synthetic rawlog of a simulated robot 
as it follows a predefined path and observes a set of landmarks (through range, and 2 direction angles).
