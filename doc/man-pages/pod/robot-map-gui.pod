=head1 NAME

robot-map-gui - Gui application for opening and editing robot-made maps (* .simplemap and * .simplemap.gz)

=head1 DESCRIPTION

B<robot-map-gui> displays simple maps depending on the config file.
It can be used to editing the robot-made maps.
